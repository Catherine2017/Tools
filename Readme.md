# 常用的python工具包
## Dir_File
处理文件和目录的相应Python工具包。
### 解压文件
1. 脚本路径：`./Dir_File/decompress.py`
2. 使用方法：
```
python ./Dir_File/decompress.py infile outpath
```

### 测试压缩文件完整性
### 检查文件md5或者生成文件md5
1. 脚本路径：`./Dir_File/file_md5.py`
2. 使用方法：
```
FileMD5:
    md5: 获取MD5信息。
    write_md5: 将MD5信息输出到文件。
md5sum: 检查文件的MD5是否和md5文件一致。
```
